Timothy John Berners-Lee created the Web.
He is an English engineer and computer scientist, best known as the inventor of the World Wide Web.
TimBL is the director of the World Wide Web Consortium(W3C)
The world's first Web-site is "info.cern.ch"
CERN is "European Institute of Particle Physics"
'Web Browser' requests information from the 'Web Server', and the Web Server that is requested provides that information to the Web Browser.
Web Browser = Client
Web Server = Server

On the Web Server side, each computer with an Internet connection is called a "host".
The computer that runs the Web Server and the company that borrows the "host" is called "Web-hosting company".
